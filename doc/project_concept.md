## 1. 프로젝트 개요

### 1.1 프로젝트명
**쉽계부(TopDownBudget)** - 약간의 SNS 기능을 포함한 처음 가계부를 쓰는 사람을 위한 쉬운 가계부

### 1.2 프로젝트 목적
기존의 가계부는 Bottom Up 형태로 내역을 한건씩 등록하던 과정때문에 가계부를 시작하는 것도 어려웠기에 Top Down 형태로 관리하고 혼자 고민하면 어디서 어떻게 관리할지, 어떻게 아낄지 막막해 가계부 작성을 포기했던 사람들을 위한 SNS 기능이 포함된 가계부

### 1.3 핵심 가치
- **프라이버시**: 공유를 선택하지 않은 기본적인 가계부 기능은 Private 형태로 원하는 사람한테만 공유하거나, 아예 혼자만 사용할 수 있음
- **편의성**: 직관적인 UI/UX와 전체부터 상세 관리로 이어지는 쉬운 가계부 작성(Top Down 형태 기록으로 기존 가계부에서 Bottom Up으로 취합되는 것과 달리 전체 수입, 지출 -> 통장, 카드 별 지출 -> 상세 지출 형태로 관리하고싶은 수준까지만 관리할 수 있음)
- **SNS 기능**: 원하는 항목이나 달의 내역을 SNS 형태로 이번 달 지출을 자랑하거나, 알뜰 소비 자랑, 알뜰하게 사는 팁을 공유하고, 가계부 쉽게 잘 관리하는 팁도 공유하는  사용하는 SNS적인 기능
- **빠른 검색**: 분류, 내용, 태그, 작성일 기반 검색 기능
